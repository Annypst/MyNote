结论：**不能这么写（如果你的需求是选中 class 同时为 `inp1` 和 `inp2` 的元素，或分别选中 `inp1` 和 `inp2` 的元素）**，`.inp1 .inp2{}` 有特定的 CSS 选择器含义，和直接选中 `inp1`、`inp2` 完全不同。

### 1. 先明确 `.inp1 .inp2{}` 的含义

这是 **后代选择器（空格分隔）**，表示：**选中「拥有 `inp1` 类的元素」内部的「拥有 `inp2` 类的后代元素」**（包括子元素、孙元素等所有后代），并非选中 `inp1` 或 `inp2` 本身。
